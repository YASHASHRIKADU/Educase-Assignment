# Educase Assignment Frontend

A React frontend project for the PopX mobile-style UI assignment.

## Tech Stack
- React
- React Router
- Vite
- Plain CSS

## Run Locally
1. Install dependencies:
   npm install
2. Start dev server:
   npm run dev
3. Build for production:
   npm run build

## Routes
- `/` Welcome screen
- `/login` Login screen
- `/register` Register screen
- `/account` Account settings screen
